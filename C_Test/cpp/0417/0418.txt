static 멤버 변수(클래스 변수)
static 멤버변수는 '클래스 변수'라고도 한다. 일반적인 멤버변수와 달리 클래스당 하나씩만 생성되기 때문에디ㅏ. 다음은 전역변수 클래스의 static변수로 선언한 예이다.

선언된 클래스의 모든 객체가 공유한다.
public으로 선언이 되면 클래스의 이름을 이용해서 호출이 가능하다.
객체의 멤버로 존재하는것이 아니다.(거의 전역변수가 된거랑 같다.)

const static멤버
const static으로 선언되는 멤버변수는 다음과 같이 선언과 동시에 초기화가 가능하다.
