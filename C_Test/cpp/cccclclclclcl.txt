클래스 서술은 두 부분으로 이루어진다.
클래스 선언: 데이터 멤버와 public 인터페이스(매서드라고 부르는)
멤버 함수를 이용하여 데이터 표현을 서술한다. 
클래스 메서드 정의: 클래스 멤버 함수가 어떻게 구현되는지를 서술한다.

클래스 선언은 클래스의 개요를 제공, 메서드 정의는 세부 사항을 제공하는것
함수 = 메서드 

클래스 설계를 위한 정의 


private,public 
private에 접근 하려면 public 멤버 함수를 사용해야한다. 
데이터 은닉 데이터에 직접 접근하지 못하게 차단하는 것
멤버함수가 요구하는 매개변수의 종류와 그것이 리턴하는 값의 종류만 알면된다.
